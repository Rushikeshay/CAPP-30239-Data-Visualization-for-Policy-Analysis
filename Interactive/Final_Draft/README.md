# Global Protest Tracker 

Rushikesh Jadhav

## Description

The Global Protest Tracker is a comprehensive database documenting major protests worldwide since 2017. It records the location, timing, scale, duration, motivations, and outcomes of protests, including government responses and triggers. By capturing both size and political context, it provides a concise, esource for analyzing global patterns of civil unrest.
